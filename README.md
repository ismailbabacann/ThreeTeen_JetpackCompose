This application is a game like blackjack.The player rolls a dice three times, aiming for a total of 13 to win. If the total exceeds 13, the player loses. However, if after three rolls the total is below 13, the final roll will be conducted by the system itself. In this application, Navigation is utilized for user interface navigation, Data transfer for managing data transfer, Scaffold, Snackbar for displaying notifications to the user etc...
